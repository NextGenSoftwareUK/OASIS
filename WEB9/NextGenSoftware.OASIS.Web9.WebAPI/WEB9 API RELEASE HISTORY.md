
# WEB9 OASIS SINGULARITY LAYER API RELEASE HISTORY

This needs to be updated whenever we do any work that will affect the WEB9 Singularity Layer API.

## 1.0.0 (28/06/26)

Initial release of the WEB9 OASIS Singularity Layer API — the network observing itself.

- Live unified status aggregator: probes WEB4 (OASIS API), WEB5 (STAR API), WEB6 (AI API), WEB7 (Symbiotic), and WEB8 (Inter-Galactic) and reports back as a single unified health/status view.
- The network observing and reporting on itself in real time — the Singularity Layer made literal.
- Provider health: aggregated view of all OASIS provider statuses across the entire stack.
- Version matrix: reports the current version of every WEB layer in a single response.
- JWT authentication via the OASIS avatar system.
- Full Swagger/OpenAPI documentation.
- CORS open policy (all OASIS APIs are open/pluggable).
- Built on top of the WEB4–WEB8 OASIS stack.
