# CV Template

A modern, print-ready resume/CV template built with React, TypeScript, and Tailwind CSS.

## Features

- Clean, professional design
- Print-optimized layout (A4)
- Fully responsive
- Built with React + TypeScript
- Styled with Tailwind CSS

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Customization

Edit `max_cv_template.tsx` and modify the `resume` object to customize:
- Personal information
- Contact details
- Skills
- Education
- Work experience
- References

## File Structure

- `max_cv_template.tsx` - Main resume component
- `main.tsx` - App entry point
- `index.html` - HTML template
- `index.css` - Tailwind imports
- `vite.config.ts` - Vite configuration
- `tailwind.config.js` - Tailwind configuration
- `tsconfig.json` - TypeScript configuration
