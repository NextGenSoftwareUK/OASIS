
# WEB10 OASIS SOURCE LAYER API RELEASE HISTORY

This needs to be updated whenever we do any work that will affect the WEB10 Source Layer API.
This file is linked to the Swagger documentation for the WEB10 Source Layer API.

----------------------------------------------------------------------------------------------------------------------------
## 1.0.0 (28/06/26)

Initial release of the WEB10 OASIS Source Layer API — the Source / WEB0 made literal.

- Root source-of-truth endpoint for the entire OASIS stack.
- Combines foundational OASIS runtime/version identity with a live unified status across WEB4-WEB9 (via the WEB9 Singularity aggregator).
- The ultimate single endpoint to understand the complete state of the OASIS at any given moment.
- OASIS runtime version and all WEB layer versions reported in a single response.
- JWT authentication via the OASIS avatar system.
- Full Swagger/OpenAPI documentation.
- CORS open policy (all OASIS APIs are open/pluggable).
- Built on top of the full WEB4–WEB9 OASIS stack.
