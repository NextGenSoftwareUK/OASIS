## 1.0.0 (28/06/26)

Initial release of the WEB7 OASIS Symbiotic Layer API — the non-invasive human/AI symbiosis layer.

- Real bio-signal DSP pipeline: EEG, HRV and GSR signal ingestion, processing and analysis endpoints.
- Consent-gated session management enforcing the Borg-Free pledge — no invasive AI without explicit, revocable avatar consent.
- Collective consciousness spaces: shared intention and attention fields across multiple avatars.
- JWT authentication via the OASIS avatar system.
- Full Swagger/OpenAPI documentation with XML doc comments.
- Exception filter with configurable generic error handling mode.
- CORS open policy (all OASIS APIs are open/pluggable).
- Built on top of the WEB4 (OASIS API) + WEB5 (STAR ODK) + WEB6 (AI layer) stack.
