## Usage

These logos are used for demonstration purposes in the POC frontend. For production use, please:

1. Review each project's brand guidelines
2. Download official brand assets from each project's website
3. Host logos locally rather than hotlinking

## Alternatives

If CryptoLogos links become unavailable, official brand asset pages:

- **Ethereum**: https://ethereum.org/en/assets/
- **Polygon**: https://polygon.technology/brand-kit

- **Base**: https://base.org (Coinbase branding)

For the POC interview demonstration, hotlinking from CryptoLogos is acceptable. For production deployment, download and host locally.
