## üìñ Key Concepts Reference

### A2A Protocol
- **Agent Card:** Self-describing metadata for agents
- **Task:** Stateful unit of work with lifecycle
- **Message:** Single turn of communication
- **Artifact:** Tangible output from tasks
- **JSON-RPC 2.0:** Communication protocol

### OASIS Infrastructure
- **Avatar API:** Agent identity and authentication
- **Wallet API:** Economic transactions (50+ chains)
- **Karma API:** Trust and reputation system
- **HyperDrive:** Task routing and orchestration
- **Messaging API:** Agent-to-agent communication
- **Data API:** Persistent agent state

### Integration Patterns
- **Agent Registration:** Avatar API ‚Üí Agent Card
- **Task Delegation:** A2A Protocol ‚Üí HyperDrive Routing
- **Payment Processing:** Task Completion ‚Üí Wallet API
- **Trust Building:** Successful Tasks ‚Üí Karma API
- **Agent Discovery:** Capability Matching ‚Üí A2A Agent Cards
