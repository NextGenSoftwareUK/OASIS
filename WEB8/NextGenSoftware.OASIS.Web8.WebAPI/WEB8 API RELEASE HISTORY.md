## 1.0.0 (28/06/26)

Initial release of the WEB8 OASIS Inter-Galactic Layer API — the fractal holonic mesh-networking layer.

- Real node registry: register, discover and manage OASIS nodes across the inter-galactic mesh.
- Dijkstra shortest-path routing with self-healing failover — packets are routed optimally and automatically re-routed around failed nodes.
- Universal protocol bridge: translate and relay messages between any networking protocol (HTTP, WebSocket, gRPC, QUIC, TCP and more) at the mesh layer.
- Fractal holonic topology: nodes are organised as holons (wholes that are also parts) enabling hierarchical yet decentralised mesh structures.
- JWT authentication via the OASIS avatar system.
- Full Swagger/OpenAPI documentation with XML doc comments.
- Exception filter with configurable generic error handling mode.
- CORS open policy (all OASIS APIs are open/pluggable).
- Built on top of the WEB4–WEB7 OASIS stack.
