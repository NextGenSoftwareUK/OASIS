## 1. WEB4 OASIS API - Data Aggregation & Identity Layer

The **WEB4 OASIS API** gives you universal data aggregation and identity management across Web2 and Web3.

**Use it for:** Identity management, data storage, wallet operations, NFT management, and cross-chain operations. It supports multiple chains; see the [Supported Providers](web4-oasis-api/overview.md#supported-providers) page for the full list.

### Key APIs

| API | Description | Endpoints |
|-----|-------------|-----------|
| [Avatar API](web4-oasis-api/authentication-identity/avatar-api.md) | Complete user management system with 80+ endpoints | `/api/avatar` |
| [Wallet API](web4-oasis-api/blockchain-wallets/wallet-api.md) | Multi-chain wallet support for 50+ blockchains | `/api/wallet` |
| [NFT API](web4-oasis-api/blockchain-wallets/nft-api.md) | Cross-chain NFT operations with GeoNFT support | `/api/nft` |
| [Data API](web4-oasis-api/data-storage/data-api.md) | Universal data operations with auto-replication | `/api/data` |
| [Holons API](web4-oasis-api/data-storage/holons-api.md) | Load, save, and manage holons (core data model) | `/api/data` |
| [Karma API](web4-oasis-api/authentication-identity/karma-api.md) | Reputation and reward system | `/api/karma` |
| [HyperDrive API](web4-oasis-api/network-operations/hyperdrive-api.md) | Auto-failover system with intelligent routing | `/api/hyperdrive` |

**Key Features:**
- ✅ **50+ Blockchain Support** - Ethereum, Solana, Polygon, Arbitrum, and more
- ✅ **100% Uptime** - HyperDrive auto-failover system
- ✅ **Universal Identity** - Avatar system with SSO
- ✅ **Cross-Chain NFTs** - Multi-chain NFT collections
- ✅ **Auto-Replication** - Data redundancy across providers

[Get Started with WEB4 OASIS API →](web4-oasis-api/overview.md)

## 2. WEB5 STAR API - Gamification & Metaverse Layer

The **WEB5 STAR API** provides advanced functionality for building immersive metaverse experiences.

**Use it for:** Game development, metaverse creation, quest systems, and virtual world building. Optimized for high-volume operations, dashboards, and data-heavy applications.

### Key APIs

| API | Description | Endpoints |
|-----|-------------|-----------|
| [Missions API](web5-star-api/game-mechanics/missions-api.md) | Complete mission and quest management | `/api/missions` |
| [GeoNFTs API](web5-star-api/location-services/geonfts-api.md) | Location-based NFTs for AR/VR | `/api/geonfts` |
| [OAPPs API](web5-star-api/development-tools/oapps-api.md) | Application management and publishing | `/api/oapps` |
| [Celestial Bodies API](web5-star-api/celestial-systems/celestial-bodies-api.md) | Virtual world objects and spaces | `/api/celestialbodies` |
| [Quests API](web5-star-api/game-mechanics/quests-api.md) | Quest creation and management | `/api/quests` |
| [Inventory API](web5-star-api/data-structures/inventory-api.md) | Virtual item management | `/api/inventoryitems` |

**Key Features:**
- ✅ **Low/No Code** - Visual OAPP builder
- ✅ **Cross-Platform** - Works across all engines
- ✅ **Version Control** - Built-in publishing system
- ✅ **Template Library** - Pre-built components
- ✅ **Real-Time Sync** - Live collaboration

[Get Started with WEB5 STAR API →](web5-star-api/overview.md)
