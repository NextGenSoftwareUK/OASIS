# OASIS Major Milestones: 2011 to Present

## Overview

OASIS (Open Advanced Secure Interoperable Scalable-System) has evolved from an early vision in 2011 into a comprehensive Web4/Web5 infrastructure that unifies all Web2 and Web3 technologies. This document outlines the major milestones in OASIS development.

---

## Timeline of Major Milestones

### **2011 - The Beginning: R&D and Vision**
- **Initial R&D Phase Begins**: David Ellams (Anorak) starts research, design, and prototyping for "Our World" - the digital twin of our planet
- **7 Years of Planning**: Foundation laid for what would become the OASIS platform
- **Early Conceptualization**: Vision for a universal API to connect everything to everything begins to take shape

### **2017 - Public Launch and Vision**
- **Founders Introduction**: Public introduction of the Our World/OASIS vision (May 2017, remastered November 2017)
- **Initial Vision and Architecture Design**: Core architectural concepts and design principles established
- **Community Building Begins**: Early community engagement and social media presence established

### **2018 - Partnerships and Prototypes**
- **UN Partnership**: Partnership with Be Earth UN-IGO (United Nations Intergovernmental Organisation)
- **Buckminster Fuller World Peace Game**: Integration of Buckminster Fuller's World Peace Game concepts
- **First Smartphone Prototype**: Early prototype development begins for AR version
- **Hardware Concepts**: NextGen hardware concepts and open-source device discussions begin
- **Blog Posts and Updates**: Regular community updates and vision sharing

### **2019 - Core Development Acceleration**
- **Avatar Karma System Development**: Digital twin of users (Avatar Karma System) development begins
- **Holochain Integration**: Deep integration with Holochain ecosystem begins
- **OASIS API Goes Open Source**: Major milestone - OASIS API becomes open source (August 2019)
- **HoloNET Development**: Core .NET HDK and HoloNET development accelerates
- **Core API Development**: Foundation for universal API architecture established
- **Prototype Updates**: Significant progress on smartphone and AR prototypes

### **2020 - WEB4 OASIS API Implementation**
- **Anorak's Journey Video Series**: David Ellams begins documenting the creation of OASIS (October 2020)
- **WEB4 OASIS API Implementation**: Active development and testing of WEB4 OASIS API
- **Provider Integration**: Integration of multiple Web2 and Web3 providers begins
- **Architecture Refinement**: Core architecture design continues to evolve

### **2021 - STAR ODK Birth and Expansion**
- **STAR ODK Born**: STAR (Omniverse Interoperable Metaverse Low Code Generator) born on Lions Gate (August 8, 2021)
- **Digital Twin Expansion**: Evolution from digital twin of planet to digital twin of Solar System
- **WEB5 STAR ODK Development**: Active development of STAR ODK begins
- **Metaverse Integration**: Metaverse and gamification layer development accelerates
- **Full Ecosystem Deployment**: Community building and ecosystem deployment begins

### **2022 - Continued Development**
- **Provider Expansion**: Continued integration of 50+ providers across Web2 and Web3
- **STAR Development**: Continued development of STAR ODK and metaverse capabilities
- **Community Growth**: Global community of developers, users, and contributors expands

### **2023 - Production Releases**
- **Production-Ready Releases**: First production-ready releases of WEB4 OASIS API
- **STAR ODK v2.2.0 Released**: Major STAR ODK release (November 2023)
- **OASIS Runtime v3.3.1 Released**: Runtime system updates (November 2023)
- **Advanced Features**: Implementation of advanced features and capabilities

### **2024 - Major Updates and Releases**
- **HoloNET v3.0.2 Released**: Complete rewrite from the ground up
  - Compatibility with latest Holochain beta
  - Upgrade to .NET v8.0
  - HoloNET ORM package introduction
  - New HoloNET Manager UI
  - Full Holochain Conductor Admin API implementation
  - HyperNET feature for P2P networking
- **STAR Development**: Continued advancement of STAR ODK capabilities
- **Style and StarNet Development**: New features for NFT collections and quest management
- **100K+ Lines of Code**: Massive development effort with over 100,000 lines of code written

### **2025 - STAR Demos and MVP Release**
- **STAR DEMO Series**: STAR demo series launched on Solstice (December 21, 2024)
- **STAR MVP Release**: MVP release as Christmas present to community (December 2024)
- **Guardians of Infinite Reality NFT Campaign**: NFT campaign launch (August 2025)
- **STAR ODK v2.2.0+**: Continued releases and improvements
- **Community Celebrations**: Year of growth and excitement with big launches ahead

## Key Evolution Highlights

### **From Concept to Reality**
- **2011-2017**: Vision and R&D phase
- **2017-2020**: Initial vision and architecture design
- **2019-2022**: Core API development and provider integration
- **2020-2023**: WEB4 OASIS API implementation and testing
- **2021-2025**: WEB5 STAR ODK development and metaverse integration
- **2021-Present**: Full ecosystem deployment and community building

### **Technology Integration**
- Successfully integrated **50+ Web2 and Web3 providers**
- Pioneered **auto-failover technology** (OASIS HyperDrive)
- Developed **universal data aggregation** capabilities
- Created **hot-swappable provider architecture** with zero vendor lock-in

### **Community Growth**
- Built a **global community** of developers, users, and contributors
- Established partnerships with **UN organizations** and other entities
- Created **open-source ecosystem** with active development
- Regular **community updates** and engagement

### **Innovation Leadership**
- **First universal API** that connects everything to everything
- **Revolutionary multi-layer NFT system** (Web3, Web4, Web5)
- **GeoNFTs** for location-based digital assets
- **StarNet Platform** for publishing applications
- **Low-code/no-code metaverse development** with STAR ODK

---

## Current Status (2025)

OASIS has evolved from its early R&D phase in 2011 into a comprehensive Web4/Web5 infrastructure that:

- **Unifies all Web2 and Web3 technologies** into a single intelligent system
- **Provides universal API** for seamless integration
- **Offers auto-failover technology** ensuring 100% uptime
- **Supports 50+ providers** across multiple categories
- **Enables metaverse development** through STAR ODK
- **Fosters global community** of developers and users

The journey from 2011 to 2025 represents over 14 years of research, development, and innovation, transforming the vision of a universal API into a working, production-ready system that revolutionizes how we interact with the internet.
