# OASIS Sync with LFG — February 20, 2025

**Recording:** 72 mins — https://fathom.video/share/B3b7oQkHgzadtbxxLu13MwGsCnkCR4JL  
**Participants:** Max Gershfield (OASIS), Saif Amer (Laissez Faire Group), Cordy Joseph

---

## Summary (key outcomes)

- **OASIS Commercial vs Oasis Main:** Max explained need for autonomy and a licensed commercial vehicle; Saif said what matters is **where the IP is housed**. OASIS Commercial currently owns nothing — Saif flagged that as a problem for clawbacks and deal structure. Can use Oasis Main (NextGen World Limited) as the counterparty instead.
- **IP / licensing:** LFG need clarity on what IP exists, who owns it, licence types (no copyleft forcing B2G code open), CLA/contributor IP, and SBOM for integrations (e.g. Pose). Corporate structure confirmed: UK limited company, Next Gen World Limited; IP resides there.
- **Fundraising:** Fundraising **cannot** be part of LFG’s scope for this deal (backlog; only ~4 projects per year). Introductions only. Max agreed partners/pilots/integrations are the priority.
- **Token / TGE:** Park for now. Saif’s main concern: no token launch without LFG alignment; otherwise they’d have to withdraw. Vesting terms premature until tokenomics and use cases are set.
- **Two-phase structure (Saif’s proposal):**  
  - **Phase 1 (low friction):** Simple partnership agreement. Good-faith introductions (e.g. Pose), GTM support, no big upfront equity ask. Small clawbackable equity/token float pegged to GTM execution if needed. SBOM and legal clarity (no copyleft, no IP risk) required for B2B integrations.  
  - **Phase 2 (contingent on Phase 1):** B2G facilitation. **No baseline equity** from Oasis for B2G; LFG earn equity/token float per deal (e.g. per pilot closed, or per $ deal value vs valuation). JVs are per country, case-by-case (e.g. 40% in Ghana spearhead); revenue share in JVs. Phase 2 numbers to be set when Phase 1 has gone well. Saif indicated B2G equity float in 15–17.5% range when it happens.
- **Immediate next steps:** (1) Provide fully diluted cap table, option pool, warrants, token treasury correlation. (2) Clarify licence types (GPL/copyleft or not), CLA/contributor IP. (3) SBOM for core OASIS/Holonic Braid for Pose or other B2B integrations. (4) Optional: simple partnership agreement + side channel for intro blurb; or Phase 1 deal with small clawbackable float + Phase 2 agreement. (5) Max to send outline for Holonic Braid + OpenServe + LFG three-way conversation.
- **Holonic Braid / OpenServe:** Max to put together outline; Saif/Leo review; if green light, three-way call to explore proving and piloting Holonic Braid (UAE government angle) with OpenServe and LFG.
- **Security / audits:** Third-party audit pre-deployment for serious B2G; indemnification against third-party IP claims. Reptile in place for vulnerability scanning; David controls master branch.
