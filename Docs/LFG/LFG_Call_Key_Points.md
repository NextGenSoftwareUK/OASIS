## Two-Phase Deal (Saif’s Proposal)

### Phase 1 — Low friction

- Simple **partnership agreement**.
- Good-faith **introductions** (e.g. Pose), GTM support, **no big upfront equity ask**.
- Small **clawbackable** equity/token float pegged to GTM execution if needed.
- **Required for B2B:** SBOM + legal clarity (no copyleft, no IP risk) for integrations (e.g. Pose).

### Phase 2 — Contingent on Phase 1

- **B2G facilitation.** No baseline equity from OASIS for B2G; LFG **earn** equity/token per deal (e.g. per pilot closed, or per $ deal value vs valuation).
- **JVs** are per country, case-by-case (e.g. 40% in Ghana spearhead); revenue share in JVs.
- Phase 2 numbers to be set when Phase 1 has gone well. Saif indicated B2G equity float in **15–17.5%** range when it happens.
