## Glif-Specific Data Visualization Guidelines

### Chart Types:
- **Bar Charts**: For storage, cost comparisons
- **Line Charts**: For network complexity, performance over time
- **Stacked Charts**: For cost breakdowns
- **Tables**: For multi-metric comparisons

### Color Scheme:
- **Conventional**: Red/Gray (#E74C3C, #95a5a6) - Problems, inefficiency
- **Holonic**: Blue/Purple (#4A90E2, #7B68EE) - Solutions, efficiency
- **Efficiency Callouts**: Green (#2ECC71) - Positive metrics

### Data Visualization Best Practices:
- Use log scale for large number ranges
- Clear axis labels
- Prominent efficiency ratios
- Visual emphasis on key numbers
- Consistent color coding
